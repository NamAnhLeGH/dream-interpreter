# Dream Interpreter

AI-powered dream interpretation application with user authentication, dream journal, and recurring symbol analysis.

## Setup

### Backend
```bash
cd backend
npm install
cp env.template .env
# Edit .env with your database URL and JWT_SECRET
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features

- AI-powered dream interpretation
- User authentication (register/login)
- Dream journal with filtering by symbols
- Recurring symbol tracking
- Admin dashboard with analytics
